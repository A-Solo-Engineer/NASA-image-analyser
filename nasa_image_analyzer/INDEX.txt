================================================================================
                     CSV OUTPUT DETAILS
================================================================================

File: output/nasa_image_data.csv

Format: CSV with headers
Encoding: UTF-8
Records: 1 per analyzed image

Columns (18+):
  • title - Image title
  • date - Publication date
  • explanation - Scientific description
  • image_file - Local filename
  • image_url - Original URL
  • image_width - Width in pixels
  • image_height - Height in pixels
  • image_size_pixels - Total pixels
  • brightness - Brightness (0-255)
  • brightness_category - Dark/Medium/Bright
  • mean_red - Red channel average
  • mean_green - Green channel average
  • mean_blue - Blue channel average
  • file_size_kb - File size
  • detected_keywords - Scientific terms
  • source - Copyright info
  • has_hd_version - HD available (yes/no)
  • analysis_timestamp - When analyzed

Open in Excel, Google Sheets, or any spreadsheet app!

================================================================================
                      CONFIGURATION OPTIONS
================================================================================

Edit config.py to customize:

MAX_IMAGES = 100
  → Number of images to fetch (default: 100)
  → Adjust for more/fewer images

CONFIDENCE_THRESHOLD = 0.5
  → Detection confidence (0.0-1.0)
  → Higher = stricter, more accurate

REQUEST_TIMEOUT = 30
  → API timeout in seconds
  → Increase for slow connections

RETRY_ATTEMPTS = 3
  → How many times to retry failed requests
  → Increase for unreliable connections

RETRY_DELAY = 2
  → Seconds to wait between retries
  → Increase for rate-limited APIs

OUTPUT_DIR = Path('output')
  → Where to save results
  → Can be any valid path

================================================================================
                     TROUBLESHOOTING
================================================================================

Problem: "No module named 'requests'"
Solution: pip install -r requirements.txt

Problem: "NASA_API_KEY not found"
Solution: Set NASA_API_KEY environment variable
  PowerShell: $env:NASA_API_KEY = "your_key"
  Cmd: set NASA_API_KEY=your_key
  Or: Create .env file with NASA_API_KEY=your_key

Problem: "Connection timeout"
Solution: Edit config.py, increase REQUEST_TIMEOUT

Problem: "Permission denied"
Solution: Run from directory with write access

Problem: "No images fetched"
Solution: Check internet, verify API key, check logs

For more help:
  → See README.md Troubleshooting section
  → Check nasa_analyzer.log for details
  → Run python test_setup.py to diagnose
