PROJECT OVERVIEW
================================================================================

A fully functional Python application that:
1. Automatically fetches images from NASA archives (APOD - Astronomy Picture of the Day)
2. Downloads and analyzes each image for scientific data
3. Extracts metadata, color analysis, brightness levels, and keywords
4. Exports all results to a structured CSV file for research and analysis

Key Achievement: Fully automated workflow from data collection to analysis to export

================================================================================
CSV OUTPUT SPECIFICATION
================================================================================

File Location: output/nasa_image_data.csv

Data Fields (18+ columns):
  1. title                    - Image title
  2. date                     - Publication date
  3. explanation              - Scientific explanation
  4. image_file              - Local filename
  5. image_url               - Original URL
  6. image_width             - Width in pixels
  7. image_height            - Height in pixels
  8. image_size_pixels       - Total pixel count
  9. brightness              - Brightness value (0-255)
  10. brightness_category    - Dark/Medium/Bright
  11. mean_red               - Red channel average
  12. mean_green             - Green channel average
  13. mean_blue              - Blue channel average
  14. file_size_kb           - File size in KB
  15. detected_keywords      - Scientific terms found
  16. source                 - Copyright/attribution
  17. has_hd_version         - HD available (yes/no)
  18. analysis_timestamp     - When analyzed

Format: UTF-8 encoded CSV with headers
Records: One per analyzed image (up to MAX_IMAGES)

================================================================================
CONFIGURATION OPTIONS
================================================================================

Edit config.py to customize:

  MAX_IMAGES                  Default: 100
    → Number of images to fetch from NASA

  IMAGE_ANALYSIS_MODEL        Default: 'yolov8n'
    → Model name for future object detection

  CONFIDENCE_THRESHOLD        Default: 0.5
    → Detection confidence level (0.0-1.0)

  REQUEST_TIMEOUT            Default: 30 seconds
    → API request timeout duration

  RETRY_ATTEMPTS             Default: 3
    → Number of retry attempts on failure

  RETRY_DELAY                Default: 2 seconds
    → Wait time between retries

  OUTPUT_DIR                 Default: 'output'
    → Where to save results

TROUBLESHOOTING QUICK REFERENCE

  → pip install -r requirements.txt

  → Increase REQUEST_TIMEOUT in config.py

Permission denied error
  → Run from directory with write permissions

Rate limit exceeded
  → Use personal API key instead of DEMO_KEY

No images fetched
  → Check internet connection
  → Verify API key is valid
  → Check nasa_analyzer.log
