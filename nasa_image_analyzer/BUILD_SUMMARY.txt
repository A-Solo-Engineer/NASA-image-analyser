================================================================================
                    WHAT THIS SOFTWARE DOES
================================================================================

1. FETCHES IMAGES
   ├─ Connects to NASA APOD (Astronomy Picture of the Day) API
   ├─ Retrieves image metadata (title, date, explanation, URLs)
   ├─ Handles connection errors with automatic retry
   └─ Validates and filters data

2. DOWNLOADS IMAGES
   ├─ Downloads image files from NASA servers
   ├─ Caches images to avoid re-downloading
   ├─ Handles network timeouts gracefully
   └─ Validates downloaded files

3. ANALYZES IMAGES
   ├─ Opens images with PIL/Pillow library
   ├─ Calculates image dimensions (width × height)
   ├─ Analyzes colors (mean RGB values)
   ├─ Calculates brightness (0-255 scale)
   ├─ Classifies brightness (Dark/Medium/Bright)
   ├─ Extracts scientific keywords from descriptions
   └─ Compiles all metrics

4. EXPORTS TO CSV
   ├─ Formats data for spreadsheet
   ├─ Creates CSV file with headers
   ├─ Writes one image per row
   ├─ Includes 18+ data columns
   └─ Uses UTF-8 encoding

5. PROVIDES LOGGING
   ├─ Logs all operations to file
   ├─ Displays real-time progress
   ├─ Records errors for debugging
   └─ Helps troubleshoot issues

================================================================================
                    CSV OUTPUT SPECIFICATION
================================================================================

File: output/nasa_image_data.csv
Format: CSV with headers
Encoding: UTF-8
Records: One per analyzed image

Sample Columns (18+):
  • title - Image title
  • date - Publication date
  • explanation - Scientific description
  • image_file - Local filename
  • image_url - Original URL
  • image_width - Width in pixels
  • image_height - Height in pixels
  • image_size_pixels - Total pixel count
  • brightness - Brightness value (0-255)
  • brightness_category - Dark/Medium/Bright
  • mean_red - Red channel average
  • mean_green - Green channel average
  • mean_blue - Blue channel average
  • file_size_kb - File size in kilobytes
  • detected_keywords - Scientific terms found
  • source - Copyright/attribution
  • has_hd_version - HD version available (yes/no)
  • analysis_timestamp - When analyzed

Example Row:
  "Helix Nebula","2023-01-15","A beautiful spiral galaxy...",
  "a1b2c3d4e5f6.jpg","https://apod.nasa.gov/...",
  1920,1080,2073600,145.32,"Medium",156.45,148.92,138.67,
  425.30,"galaxy; nebula; star","© NASA/JPL-Caltech","True",
  "2023-01-20T14:30:45"
