================================================================================
                        EXECUTION FLOW
================================================================================

1. STARTUP
   ├─ Initialize configuration
   ├─ Create output directories
   └─ Setup logging

2. DATA COLLECTION
   ├─ Connect to NASA API
   ├─ Fetch image metadata (~100 images by default)
   └─ Validate data received

3. IMAGE DOWNLOAD
   ├─ For each image:
   │  ├─ Download image file
   │  ├─ Cache locally (skip if already downloaded)
   │  └─ Handle errors gracefully
   └─ Continue with next image on failure

4. IMAGE ANALYSIS
   ├─ For each downloaded image:
   │  ├─ Open with Pillow (PIL)
   │  ├─ Analyze technical properties:
   │  │  ├─ Width × Height
   │  │  ├─ Pixel count
   │  │  └─ File size
   │  ├─ Calculate color data:
   │  │  ├─ Mean Red value
   │  │  ├─ Mean Green value
   │  │  └─ Mean Blue value
   │  ├─ Calculate brightness (0-255)
   │  ├─ Classify brightness (Dark/Medium/Bright)
   │  └─ Extract keywords from description
   └─ Compile results

5. DATA PROCESSING
   ├─ Flatten data structures
   ├─ Format values for CSV
   ├─ Convert types (lists→strings, etc.)
   └─ Prepare headers

6. CSV EXPORT
   ├─ Create output/nasa_image_data.csv
   ├─ Write headers (18+ columns)
   ├─ Write data rows (1 per image)
   └─ Close file

7. COMPLETION
   ├─ Generate summary statistics
   ├─ Log completion
   └─ Display results path
