TROUBLESHOOTING CHECKLIST
================================================================================

PROBLEM: "No module named 'requests'"
  ☐ Check: requirements.txt is present
  ☐ Run: pip install -r requirements.txt
  ☐ Wait for install to complete
  ☐ Run: python test_setup.py
  ☐ Should pass now

PROBLEM: "NASA_API_KEY not found"
  ☐ Check: Environment variable is set
  ☐ Run in PowerShell: echo $env:NASA_API_KEY
  ☐ Should show your API key
  ☐ If empty, re-set using one of the methods above
  ☐ Try again

PROBLEM: "Connection timeout"
  ☐ Check: Internet connection is working
  ☐ Check: NASA API is online (https://api.nasa.gov/)
  ☐ Edit config.py
  ☐ Change: REQUEST_TIMEOUT = 60
  ☐ Try again

PROBLEM: "Permission denied" error
  ☐ Check: You have write access to folder
  ☐ Check: output/ folder is not read-only
  ☐ Right-click output/ → Properties
  ☐ Remove "Read-only" if checked
  ☐ Try again

PROBLEM: "No images fetched"
  ☐ Check: Internet connection
  ☐ Check: API key is valid and active
  ☐ Check: nasa_analyzer.log for error details
  ☐ Try: python test_setup.py
  ☐ Look for [OK] on API test line

PROBLEM: "ValueError" or "TypeError"
  ☐ Check: requirements.txt is installed correctly
  ☐ Run: pip install --upgrade -r requirements.txt
  ☐ Check: nasa_analyzer.log for full error
  ☐ If still failing, contact support

Troubleshooting Complete: ✓
